## 📘 Descripción

Este repositorio contiene un **manual completo para el diseño y fabricación de circuitos impresos (PCB) mediante fresado CNC**, enfocado en la estandarización de procesos dentro del laboratorio.

El documento guía al usuario desde:

* Fundamentos de CNC y G-Code
* Uso del software de control (Candle)
* Diseño electrónico en KiCad
* Generación de archivos de manufactura
* Procesamiento CAM con FlatCAM
* Ejecución del mecanizado

El manual está diseñado para **usuarios sin experiencia previa**, con un enfoque práctico y progresivo.

## 🧪 Aplicación del manual

Este manual fue desarrollado para el equipo:

* CNC 1610
* Controladora Woodpecker (GRBL)
* Software Candle
* Flujo KiCad → FlatCAM → CNC

Incluye un ejemplo práctico basado en un **oscilador astable con NE555**, que permite validar el proceso completo de diseño y fabricación.

---

## ⚠️ Consideraciones

* El equipo CNC no cuenta con paro de emergencia físico
* El control del spindle es manual
* Se recomienda ejecutar pruebas en vacío antes de mecanizar
